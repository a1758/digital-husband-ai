# 💞 Digital Husband AI

A real-time AI virtual companion powered by:

- DeepSeek-V4-Pro (LLM brain)
- Xiaomi Voice (TTS / STT)
- Live2D (emotional avatar)
- Claude Code (development assistant)

---

# 🎯 Vision

To build a deeply emotional, immersive AI companion that feels alive:

- 🧠 Long-term memory
- 🗣️ Voice conversation (real-time)
- 💬 Emotional interaction
- 🎭 Live2D animated avatar
- ❤️ Continuously evolving relationship

---

# ✨ Features

## 🧠 Emotional AI
The AI understands and reacts to:

- Mood
- Emotional state
- Relationship history
- Affection level

---

## 📚 Long-Term Memory
The system remembers:

- Birthdays
- Anniversaries
- Favorite things
- Emotional events
- Personal preferences

---

## 🗣️ Real-time Voice
Supports:

- Speech-to-text (STT)
- Text-to-speech (TTS)
- Low-latency streaming
- Interrupt handling

---

## 🎭 Live2D Avatar
Includes:

- Blinking animation
- Lip sync
- Emotion-based expressions
- Idle motion system
